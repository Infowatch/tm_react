Class: Component

Defined in: src/scripts/core/components/base.coffee

Overview

Base component class, the prototype of all project components.

Examples:

Component declaring 

class Dropdown extends App.Component
  template: App.JSX.dropdown
  locals: ->
    msg: "hello"

Component usage 

Dropdown = require "dropdown"
Dropdown

  # Properties
  text: "my button"
  onClick: ->
    alert("click")

  # Children
  [
    Link to: "users"   # children also may be passed
    Link to: "events"  # via properties with `children` key.
  ]                    # If children were passed in 3-rd param -
                       # props.children will be ignored
